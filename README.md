# EMPTY VOID

## Overview
EMPTY VOID is a 2D platformer action game inspired by Hollow Knight, built using Godot 4. Dive into a beautifully crafted world filled with challenging enemies, intricate platforms, and a captivating story.

## Features
- Stunning visuals and animations inspired by Hollow Knight.
- Engaging gameplay mechanics that challenge your skills.
- An immersive storyline with unique characters and lore.
- Explore a large, interconnected world filled with secrets.

## Installation
To install and run EMPTY VOID, follow these steps:
1. Download the latest version from the [releases page](#).
2. Extract the downloaded files.
3. Open the project in Godot 4.
4. Run the game.
